Workshop: Node.js Performance training

One day hands on training about Node.js Performance brought to you by YLD! and delivered by Tom Gallacher, Hassy Veldstra and Igor Soarez.

Node.js is great for real-time services, but what do you do when performance starts lacking? This workshop will help you gain the knowledge to identify, troubleshoot and improve performance issues in Node.js applications.

Come if you are interested in

  • Load testing services
  • Using tools to monitor performance
  • Finding performance bottlenecks
  • Troubleshooting performance problems
  • Profiling an application and improving response time

The workshop covers

  • How to define and measure the performance of your app
  • Load generation and benchmarking
  • Using tools and Tools and techniques for monitoring
  • Understanding the event loop and measuring event loop lag
  • Profiling V8 CPU usage
  • How the GC works and its impact on performance
  • Using heapdumps to analyze V8 memory
  • Fixing I/O-bound, CPU-bound, and memory-bound processes
  • Techniques to improve performance of hot code paths

Who should attend

Developers who wish to extend their understanding of Node.js and learn how to develop performant applications.

Pre-requisites

  • Experience with server side application development
  • Experience with Linux or other Unix based systems
  • Intermediate knowledge of Node.js
  • Intermediate understanding of operating systems
